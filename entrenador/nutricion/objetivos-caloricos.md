# Objetivos Calóricos y Macros

> ⚠️ Valores estimados con fórmula Mifflin-St Jeor. Pendiente confirmar con nutriólogo.

## Cálculo base

| Métrica | Valor | Fórmula |
|---|---|---|
| TMB | ~1,892 kcal/día | Mifflin-St Jeor: 10×100 + 6.25×170 − 5×35 + 5 |
| Factor actividad | 1.375 | Actividad ligera-moderada (bicicleta + gym) |
| TDEE | ~2,600 kcal/día | TMB × factor |
| Déficit objetivo | ~400 kcal/día | Déficit moderado para recomposición |
| **Calorías objetivo** | **~2,200 kcal/día** | TDEE − déficit |

## Macros objetivo

| Macro | Objetivo | Base de cálculo |

## Referencia del nutriólogo

El usuario recuerda haber recibido como referencia ~1,800 kcal/día.
Eso implicaría un déficit de ~800 kcal (agresivo). Pendiente verificar.

**Mientras tanto:** usar 2,200 kcal como objetivo hasta confirmar.
